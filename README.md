# gadget Collection Tool project using Spring Boot Kotlin

This is the source code for a Spring Boot application developed with Kotlin
and Spring Data JPA.

Launch the application:

```sh
  ./gradlew bootRun
```

Run tests:

```sh
  ./gradlew test
```
